# 组合七：社群共创型

**适用特征**：A1 个人IP × B2 B2C大众（粉丝/追随者）× C4 社群氛围 + C2 故事共鸣

**适用场景**：KOL/博主/内容创作者、兴趣社群主理人

**典型明暗线**：
- 明线：加入社群、关注/订阅
- 暗线：真实连接、不再孤独、找到同类

**设计必须**：
- 强烈的个人风格，页面是创作者自己的世界
- 社群活力感（成员、活动、内容的生命力，让人想"进去"）
- 低门槛参与感（"加入"比"购买"感觉更轻）
- 真实感优于精美感，不需要过度打磨

**设计避免**：
- 企业感、正式感、权威感
- 过于规整的标准设计
- 缺少社群内容预览（空洞的描述）

**色系约束**：不怕用高饱和度和个性色，颜色要与创作者人格/内容风格强关联，可以打破行业惯例

**字体约束**：个性感优先，可混搭，甚至可加手写体，字体有创作者的审美印记

**布局约束**：内容瀑布流、照片墙、社群 preview，布局节奏活泼，动态感强

**推荐风格搭配**：S6 编辑杂志 / S9 活力多彩

**校准案例**：待补充（参考：小红书博主/B站UP主个人站、有赞/知识星球主理人）
