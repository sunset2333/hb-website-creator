# 组合五：知识专家型

**适用特征**：A1 个人IP × B2 B2C大众（学习者/职场人）× C5 专业权威 + C3 内容展示

**适用场景**：教育培训、职业咨询、知识付费、商业顾问

**典型明暗线**：
- 明线：付费咨询、课程购买、方案定制
- 暗线：找到志同道合的朋友、帮助中小企业少走弯路
- 参考：蒋苏 — 明线(付费咨询) / 暗线(做对的事，比把事做对更重要 + 找同行者)

**设计必须**：
- 凸显个人专业背景（学历/经历/媒体背书/服务过的客户）
- 内容可信度感（引用数据、展示学员成果、媒体报道）
- 清晰的行动指引，课程购买/咨询预约路径短而清晰
- IP 与知识体系的强关联（"学XX就找XX"的心智）

**设计避免**：
- 定位模糊、情感化但缺少专业支撑
- 页面信息过载、什么都有却没有重点

**色系约束**：主色稳重有力（深蓝/墨色/深绿），点缀色有活力感（亮橙/亮绿/金），传递"专业但不刻板"

**字体约束**：标题厚重有力，正文清晰易读，可用一款有个性的展示字体做标题

**布局约束**：个人权威区（照片+资历）要醒目，课程结构清晰，学员成果用数据或引用突出

**推荐风格搭配**：S2 企业专业 / S7 便当网格

**校准案例**：蒋苏私域站（[cases/006-jiangsu.md](../cases/006-jiangsu.md)）
