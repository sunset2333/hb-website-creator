# #013 伙伴云

## 基础档案
- **主体**：A2（品牌公司）| **受众**：B1（创业者/OPC一人公司/主理人）| **信任**：C1数据+C2故事+C3案例+C5权威
- **组合**：自定义 · AI创业Agent型（近似组合二科技专业型，但增加了暗线温度）
- **页数**：2页（index/pricing）| **技术**：现代框架站（React/Vue）
- **色系**：四色渐变——紫`#8B5CF6` + 蓝`#38BDF8` + 粉`#EC4899` + 橙`#F59E0B`
- **设计关键词**：渐变丰富、便当网格、明暗线策略、智能推荐、复杂定价
- **完成时间**：2026-03

## 明暗线
- **明线**：AI创业合伙人（台面/正式） — 用于标题、品牌定位、SEO、CTA
- **暗线**：创业搭子（一对一/温情） — 用于副标题、功能描述、交互细节、Footer
- **桥梁词**："伙伴"（明线：合伙人 / 暗线：搭子、朋友）
- **收尾金句**："创业不孤单，我有伙伴云"
- ✅ **案例库中明暗线设计最完整的项目** — 在商业定位文档中系统定义了明暗线策略、使用场景、话术转换

## 设计决策复盘
- **选对了**：
  - 明暗线设计是全站灵魂——明线保证B端专业感，暗线传递陪伴温度，两者通过"伙伴"桥梁词无缝衔接
  - 四色渐变体系（紫/蓝/粉/橙）传递AI原生、进化、多维特性，与单色暗黑科技站形成差异
  - Bento Grid便当卡片展示三端能力（找方向/做交付/管经营），信息立体
  - 独立定价页：五版本+三种加油包+企业级部署+交互式智能推荐工具（4题问卷→推荐结果+费用估算）
  - 8个跨行业案例证明通用性
- **踩过的坑**：定价页功能明细表在小屏幕可能过度压缩；旧版定价页（site-v1）仍存在可能造成用户困惑
- **可复用模式**：
  - **明暗线策略框架**（所有项目的标杆参考）
  - 智能推荐工具（问卷→推荐→费用估算，适合多版本SaaS定价）
  - 便当网格（Bento Grid）展示多维能力
  - 四色渐变品牌系统（适合AI/科技/创新类产品）
