# #006 蒋苏私域站

## 基础档案
- **主体**：A1（纯个人IP）| **受众**：B1（200+中小企业）| **信任**：C2故事+C3案例+C5权威
- **组合**：自定义 · 草根实战型
- **页数**：5页SPA（React CDN，多文件组件架构）| **技术**：React 18 CDN + Babel Standalone
- **色系**：深青`#2B4C58` + 暖琥珀`#C47D3B` + 暖米白`#F7F5F2`
- **设计关键词**：真实、质朴、草根、朋友感、实战
- **完成时间**：2026-03

## 明暗线
- **明线**：付费咨询（加企微/打电话）
- **暗线**：找志同道合的朋友 — "做对的事，比把事做对更重要"
- **桥梁词**："同行"（明线：同行业伙伴 / 暗线：一起走的人）
- 注：暗线在About页11节成长时间线和全站金句中有自然渗透，但未作为独立策略系统规划

## 设计决策复盘
- **选对了**：11节个人时间线极大增强"真人感"；3个具名案例+1个失败案例比匿名评价可信度高数倍；教学vs代建对比表直观传达服务差异；20个解决方案截图的可搜索目录让"200+客户"变得具体可感；双CTA（企微+电话）覆盖不同年龄段
- **踩过的坑**：CSS变量在8个文件中重复声明；Babel浏览器端编译有性能开销；Contact页在导航中缺失
- **可复用模式**：成长时间线（适合所有个人IP项目）；失败案例展示（罕见但极高可信度）；解决方案可搜索目录（适合方案多的顾问型IP）；教学vs代建对比表
