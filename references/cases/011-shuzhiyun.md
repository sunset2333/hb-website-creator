# #011 数智云

## 基础档案
- **主体**：A2（品牌公司）| **受众**：B1（传统行业中型企业决策者）| **信任**：C1数据+C3案例+C5权威
- **组合**：组合二 · 科技专业型
- **页数**：1页主站+6个行业案例页 | **技术**：Vue 3 + Vite + TypeScript + Tailwind CSS
- **色系**：蓝色系+专业灰（具体值待确认）
- **设计关键词**：流程再造、按效果收费、数据驱动、行业覆盖、13年深耕
- **完成时间**：2026-03

## 明暗线
- ⚠️ **未设计明暗线**。以"按效果收费"的差异化卖点为单线叙事。
- 潜在明线：咨询合作（按效果收费降低决策门槛）
- 潜在暗线：让传统行业不被数字化时代抛下、用流程再造真正帮企业提效（非卖软件）

## 设计决策复盘
- **选对了**："按效果收费，无效果不收费"是极强的信任锚点——行业罕见；三步法方法论（流程再造→绩效驱动→数字固化）清晰有记忆点；客户案例有量化成果（恒通能源营收2500万→2800万）；华新化工获国家级管理创新成果二等奖是硬核背书
- **踩过的坑**：网站设计文档缺失；创始人/核心团队背景介绍缺乏；定价区间不清晰
- **可复用模式**：按效果收费承诺展示（适合咨询/服务类企业）；多行业案例覆盖展示；量化成果对比（before→after数据）
- ⚠️ **技术栈特殊**：唯一使用Vue+Vite+TypeScript的项目，非纯静态HTML
